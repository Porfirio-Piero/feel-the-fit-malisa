# Feel the Fit by MaLisa - Website

A professional personal training and fitness studio website for Hamilton, NJ.

## Preview

This is a modern, conversion-focused fitness website featuring:
- Hero section with compelling headline
- Services overview
- Pricing packages
- Online booking integration ready
- Testimonials section
- Contact form
- Mobile-first responsive design

## Quick Start

1. Open `index.html` in a browser
2. Customize the content in `index.html`
3. Update contact information
4. Add real testimonials
5. Integrate booking system (Calendly, Acuity, etc.)

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
  --primary: #10b981; /* Green */
  --secondary: #8b5cf6; /* Purple */
  --accent: #f59e0b; /* Orange */
}
```

### Content
Update text in `index.html`:
- Business name
- Phone number
- Address
- Services
- Pricing
- Testimonials

## Deployment

### Option 1: Netlify (Free)
1. Create account at netlify.com
2. Drag & drop the folder
3. Get free URL

### Option 2: Vercel (Free)
1. Create account at vercel.com
2. Import from Git or upload
3. Get free URL

### Custom Domain
1. Buy domain ($10-15/year)
2. Point DNS to hosting
3. Update URLs in code
